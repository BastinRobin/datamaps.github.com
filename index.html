DataMaps

Interactive maps for data visualizations. Bundled into a single Javascript file.

Download datamaps-usa.js Download datamaps-world.js

World map

USA specific

$("#map_world").datamaps({scope: 'world'})
$("#map_usa").datamaps({scope: 'usa'})

Add data to create choropleth maps

$('#map_election').datamaps({
      scope: 'usa',
      popupTemplate: _.template('<div>...</div>'),
      fills: {
        'REP': '#CC4731',
        'DEM': '#306596',
        'HEAVY_DEM': '#667FAF',
        /*... snipped */
        defaultFill: '#EDDC4E'
      },
      data:{
        "AZ": {
            "fillKey": "REP",
            "electoralVotes": 5
        },
        "CO": {
            "fillKey": "LIGHT_DEM",
            "electoralVotes": 5
        } /* ... snipped */
})

 

Listen for events externally

$('#map_events').datamaps({
  //turn off hover popup
});
$('#map_events').on('map-click',
  function(event, data) {
    alert('Hello ' + data.geography.name);
});

Project Information

  • Uses D3.js
  • and Backbone, Underscore & Zepto
  • Delivered as a jQuery plugin
  • Optionally, if you use Backbone, also delivered as a standalone Backbone view
  • Available bundled with dependencies and without(for existing Backbone apps)

Available Builds

  • USA only w/ dependencies
  • USA only w/o dependencies (Backbone, _ and $ would need to be provided)
  • Countries only w/ dependencies
  • Countries only w/o dependencies
  • USA & Countries w/ dependencies
  • USA & Countries w/o dependencies

Default Options

{
      projection: 'equirectangular', // 'albersUsa'
      scope: 'world', // 'usa'
      highlightOnHover: true,
      showPopupOnHover: true,
      popupTemplate: _.template('<div class="hoverinfo"><%= geography.properties.name %></div>'),

      /* highlight defaults */
      highlightBorderColor: '#FA0FA0',
      highlightBorderWidth: 2,

      borderColor: '#FFFFFF',
      borderWidth: 1,

      /* fill settings */
      data: {},
      fills: {
        defaultFill: '#BADA55'
      }
}
